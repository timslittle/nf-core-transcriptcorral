
EvidentialGene source code changes, 2019may to 2020jan .. to 2020mar04

Major new script versions are 
  tr2aacds4.pl and evgpipe_sra2genes4v.pl
    with updates and additions of associated pipeline components.
  tr2ncrna.pl  : new pipeline to select non-coding RNA subset of input transcripts

Major component revisions
  cdna_bestorf.pl, cdna_proteins.pm : revised bestorf calculations following tests
  blasttrset2exons2.pl : new measurement script, exon chain analysis for alternate transcripts
  trclass_resolve_strandmix.pl : resolve genes w/ mixed-strand CDS alternates
  asmrna_dupfilter4.pl  : classify transcripts by self-alignment, with new,updated measures
  asmrna_altreclass4.pl : reclassify alternates and noclass using new,updated measures
  trclass2pubset.pl : revised sequence set publication
  
Useful additions
  altparclassify.pl : classify transcript set as to alternate/paralog by self-alignment
  cdshexprob.pl     : calculate CDS coding potential from transcript CDS - UTR hexamers
  cdsalignkaks.pl   : calculate KaKs statistics for CDS-alignments, 
                       of two species/pops with KaKs_Calculator and blastn
  evg_buscogenesum.pl,evg_busco2v.py, : measure BUSCO conserved genes, for Evg gene IDs
